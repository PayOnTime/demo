## Running this Demo

### Local Machine
- Open a terminal and run the following commands

    ```bash
    cd ~
    git clone https://github.com/PayOnTime/demo.git
    cd demo
    npm install
    npm start
    ```
    
- Browse to [localhost:3000](http://localhost:3000) in your browser.
